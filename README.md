# Node - Módulo 1

Aplicação usando Node.js, Express, Nunjucks, EditorConfig e ESLint.

## Resumo

Esta aplicação foi desenvolvida para realizar os seguintes procedimentos:

- Lista todos os usuários
- Adiciona novos usuários
